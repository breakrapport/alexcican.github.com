Light switch

This experiment features a realistic light switch and simulates the effect of turning off/on the light. With the help of jQuery, the brightness of the page (light) is also reduced upon clicking the switch. I wanted to push myself and create a realistic light switch, so the light, gradients, and shadows had to be spot on in order to sell the effect.

Dribbble website

This is a one page website that shows you how to make a popular shot on Dribbble. It’s more of a tongue-in-cheek, sarcastic, step-by-step rather than an accurate guide. But the guide reached its goal, and is funny because it’s true. The website was perceived well in the design community and got some attention.

Twitter button

A simple Twitter button that simulates the effect of pressing a physical button. I spent some time tweaking and refining the lighting and shadows in order to get them exactly right. No images were used; the Twitter bird is a typeface icon that scales nicely on retina devices. Updated in 2012 with the new Twitter logo.
