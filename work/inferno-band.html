Concept Idea

Create a dark, textured website for my band Inferno. The website should reflect the style of music and provide insightful details about the band, photos, videos and contact details.

Production

The website was coded in xHTML Strict 1.0 and CSS 2.1. The transparent background for the text container was created using PNGs. Because no CMS was used, all news and upcoming shows were entered manually (i.e. editing the home page and the respective page).

The content container would feature a scrollbar in it in order to scroll content without going outside the paper background container.

A skull was added to the scrollbar using JavaScript to add a bit of variety and interaction. As with all other website designs, this one features a contact form in order to be able to contact the band quickly and easily.

The design received positive feedback from various peers. It was even showcased in a number of website galleries. Obviously the downside of the design is the large file size of the website. This is due mainly because of the high quality background image and the texturised decorative elements.
