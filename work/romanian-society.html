Concept Idea

The Romanian Society in Cyprus requested me to create a simple website in which they could upload their upcoming events and news. The colours shouldn't be too bright or with political connotations, so neutral colours were selected. A basic website with large menu buttons and clear headings would be ideal. Moreover, it was appropriate to include a second version of the website in English.

Production

The website was coded in valid xHTML Strict 1.0 and CSS 2.1. The latest news and upcoming events were updated manually (i.e. you had to go and edit the homepage yourself). This wasn't a problem since I would be maintaining the website as well. They requested that it should have the colours of the Romanian flag somewhere in the website, so I decided to colour the title.

The only problem persisting in the development and maintenance of the website can be found in the gallery page. Because these were static pages, no CMS is used that could generate the thumbnails for the photos automatically, I had to create the thumbnails in Photoshop. This can become a tedious job if you have to upload many photos.

The website ranked very well in the search engines, being the first result for its title, both in English and Romanian.
