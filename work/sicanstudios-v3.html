Technical Details
Website Design and Development
Tools Used: Photoshop, Notepad, xHTML Strict 1.0, CSS 2.1, jQuery
Duration: 11 days
CMS: No
Total Size: 340kB

Concept Idea

Concerned about the mega non-success that the release of SicanStudios had (64 unique visitors in the first month), lead me to redesign the website from scratch. This was the third version of SicanStudios. I went and designed the website in a Web 2.0 look, with glossy effects gradients and shadows, popular at that time.

I thought that I should try and offer something to the visitors through my website; not use it only as a display of my works. So I created a new section, called "downloads" which featured desktop wallpapers made from my photographs and tutorials written by me.

Production

For the logo of the website, I did nothing more than to throw a font, add some Photoshop effects to it and add a tagline.

For the first time, I was using sprites to minimise HTTP requests. I used sprites to load the menu (and its hover state), thumbnails and extra stuff such as line breaks. The different section thumbnails of the portfolio were black and white but once you hovered over a thumb it became coloured.

The website was developed in Notepad using valid xHTML and CSS code. No difficulties were encountered when producing the website. The reason why it took several days to complete was because I had to prepare the photos and transform them into wallpapers (various resolution options were made available). Furthermore, I also had to write some tutorials.
