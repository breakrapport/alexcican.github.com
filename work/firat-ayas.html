Technical Details
Website Design and Development
Tools Used: Photoshop, Notepad, xHTML Strict 1.0, CSS 2.1
Duration: 2 weeks
CMS: No
Total Size: 820kB

Concept Idea

Firat Ayas is a photographer who wanted a website to share his photos with the world. He pretty much knew how he wanted the design to look: grunge with lots of texture and a photo of him for the background. I designed a static website (html) that he would later transform in dynamic using his own CMS (Content Management System). The finished website would feature the possibility to comment on each photo and generate thumbnails automatically for the previous and next photo.

Production

The website was coded in valid xHTML Strict 1.0 and CSS 2.1. No difficulties were encountered in the production stage. For the rounded corners and the transparent container background, PNG was used. It's pretty much a basic website without any innovation or clever ideas, just like the client wanted it.

The only problem I see now is the large background image (1280px by 1846px) which adds a whooping 456KB, making the website “slower”.
