Technical Details
Website Design and Development
Tools Used: Photoshop, Notepad, xHTML Strict 1.0, CSS 2.1, JavaScript
Duration: 6-7 weeks
CMS: No
Total Size: 70kB

Concept Idea

This website was one of the first I did the coding all by myself. Up until now, I was using website templates and customising them lightly. This website was created during my first year for a university project. It had to feature a clean, attractive design and focus on usability rather than filling it up with Flash elements and JavaScript. Sensing that the website was lacking something, I added a style-switcher using JavaScript to switch the background colour of the website. Furthermore, it also used to add a cookie on the user's machine to remember their choice.

Production

The website was coded in valid xHTML Strict 1.0 and CSS 2.1. Not many difficulties were encountered when producing the website. The background was cleverly built using a 4x800 repeatable image (on the X and Y axis) in order to reduce loading times. Also, the background of the menu links is built using the same method. The reason why it took so much time to finish the project was because we had to progress step by step each week following the tutor's guidance. This project received a first class mark.
