Technical Details
Website Design and Development
Tools Used: Photoshop, Notepad, xHTML Strict 1.0, CSS 2.1
Duration: 7 days
CMS: No
Total Size: 29kB

Concept Idea

Develop a simplistic website to post my work and have online presence. Actual text from the about page of this version of the website: “Amongst the many services that I offer, I try to capture the essence in a simplistic way and create something original with it. For websites, my design is simple, effective, user friendly, accessible, cross-browser compatible and search engine optimized”. Since round corners were in trend, my new website should feature them. I also considered wise (at that point) to include two other versions of the website in different languages. So in total, I had three versions of the website (one in English, one in Greek and one in Romanian). This proved quickly that it was very hard to update the website regularly, so the feature was removed in later versions of the website.

Production

For the logo of the website, I wanted to create something memorable. So the best thing I could come up with (at that point) was to unite the letter “n“ from “sican“ with the letter “d“ from “studios“. The design features a full background image (the rounded corner container) with the text from the content going on top of it. The xHTML and CSS code were developed in Notepad. After, the code was checked and ensured that it is valid.

I remember that I had a hard time creating a contact form and customising the message as I wanted. With little knowledge of PHP I managed to put together a form which I'm still using nowadays for this website (in a slightly altered version).

So, the first version of SicanStudios went live on the 10th of June 2008. During that month the website would accumulate a staggering number of unique visitors: 64!
