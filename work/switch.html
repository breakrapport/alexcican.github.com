Dribbble is the source of great inspiration. Again, a design has inspired me to go out and code it just for fun, “Saturday afternoon practice” as I call it.

This week’s inspiration was a design created by Paco Xiao and it depicts a switch with “ON”/“OFF” states.

The Switch

The switch was coded in HTML5 and CSS3. Note that it has no images! The switch adjusts for two states: on and off. Check it out:

View the switch →

There were several challenges with this project, one of it was the realism of the button. It had to look real otherwise, what’s the point. A lot of time was spent in tweaking those box-shadows (and they’re quite a lot, sorry iPad 3 users). The screws were difficult to make, depict that chrome look, and I’m still not entirely happy with them. They look like they’ve been painted white.

Another challenge was to create the button elements with as few HTML elements as possible. By utilising pseudo-classes for the screws, in the end I have only four (button, inner, b, b) HTML elements for everything.

JavaScript

I wanted to avoid using JavaScript because this was meant to be an HTML & CSS3 experiment. However, I then decided that using JS would improve the user’s experience. I used JavaScript for this project only for one cause: when you click on the switch, turn off the lights (make darkness). I experimented with CSS3 transition but the background of HTML would go in funny colours before turning black.

I also added an option to select if you didn’t want the “lights OFF” when clicked on the switch.
