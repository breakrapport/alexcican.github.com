Experiments & mini projects

These fun CSS 3 & HTML 5 experiments were created in my free time in order to practice and improve my coding skills. You might encounter some bugs, as some of the technology used is not yet adopted by all browsers. Best viewed in Google Chrome.

Light switch

This experiment features a realistic light switch and simulates the effect of turning off/on the light. With the help of jQuery, the brightness of the page (light) is also reduced upon clicking the switch. I wanted to push myself and create a realistic light switch, so the light, gradients, and shadows had to be spot on in order to sell the effect.

TimeWasted web app

TimeWasted is a web app that calculates how much time you’ve wasted from your life watching TV shows. I wanted to create a web application that would look awesome on the iPhone and iPad. Each device is served with a different stylesheet (responsive design) thus ensuring the app will look excellent on each device.

In order to make app load faster, the number of connections to the database was reduced to one. All the details are then passed using POST method to the other pages. The calculations are then made using PHP from the details included in the POST. Thumbnails of each TV show on the results page are displayed by simply removing the spaces from the title of the show.

Smart searching was also implemented. So if you type in “Heisenberg”, the show “Breaking Bad” will appear. Unfortunately, because the app works using a MySQL database, it makes it tedious to keep the application up to date with the latest shows and number of seasons.
